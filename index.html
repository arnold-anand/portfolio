Projects
Chat with PDF
"Chat with PDF" is a Streamlit-based application that enables users to have interactive conversations and retrieve information from uploaded PDF documents. The application extracts text from PDFs and converts them into vector representations using OpenAI's embeddings. It utilizes a ConversationalRetrievalChain with a language model from the LangChain library to generate responses based on document content and user interactions. Users can ask question through the user-friendly interface.

Brain Tumour Detection using DL
This project develops a deep learning model that can detect and segment brain tumours from MRI images. The project uses a dataset of 3064 MRI images from Kaggle, labelled with four classes: glioma, meningioma, pituitary, and no tumour. The project applies a efficentnet b0 model by google, which is trained for biomedical image segmentation. The project also uses data augmentation techniques to increase the dataset size and diversity. The project aims to provide a fast and reliable tool for brain tumour detection and segmentation that can help medical professionals in diagnosis and treatment planning.
